# Attribution

This file is the canonical attribution list for the public paper dataset.
All paper artifacts (manifest, extracted text, gold annotations, IAA
report) must be accompanied by the source attributions below.

## Format

Each block contains:

- Source family
- Document title
- Stable URL
- License or usage note
- Access date
- Required attribution text
